Descripción detallada

The ImageResource class represents an image as a grid of Pixel objects and allows access to all of them, using the method pixels. These pixels can then be iterated over using a for loop.

Note, no changes made to the pixels in this image affect the original image opened unless you use the method save.

Example usage:

ImageResource image = new ImageResource();
for (Pixel p : image.pixels()) {
    int red = p.getRed();
    int green = p.getGreen();
    int blue = p.getBlue();
    int average = (red + green + blue) / 3;
    p.setRed(average);
    p.setGreen(average);
    p.setBlue(average);
}
image.draw();
